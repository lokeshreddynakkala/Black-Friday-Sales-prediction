A Machine Learning project focused on **predicting customer purchase amounts during Black Friday sales** using retail transaction data.

---

# 📌 Project Overview

Black Friday sales generate massive retail transactions. Understanding purchasing behavior and predicting customer spending can help businesses optimize pricing strategies, marketing campaigns, and inventory management.

This project applies **data analysis and machine learning techniques** to predict customer purchase amounts based on demographic and product-related features.

---

# 🎯 Project Objectives

- Predict **customer purchase amount**
- Analyze **shopping behavior during Black Friday**
- Identify key factors influencing purchase decisions
- Build a **regression-based predictive model**

---

# 📊 Dataset Information

The dataset contains **retail transaction data** including:

- User ID
- Product ID
- Gender
- Age
- Occupation
- City Category
- Stay in Current City Years
- Marital Status
- Product Category
- Purchase Amount (Target Variable)

### Target Variable

Purchase → Amount spent by the customer

# 🔍 Exploratory Data Analysis

EDA helped identify patterns in customer purchasing behavior.

Key insights include:

- Customers aged **26–35** contribute the highest sales.
- **Male customers** tend to spend more than female customers.
- Certain **product categories dominate overall sales**.
- Customers from **City Category B** generate high purchase values.

Visualizations used:

- Distribution plots
- Bar charts
- Correlation analysis
- Purchase trend analysis

---

# 🧠 Machine Learning Models Used

Several regression models were trained and compared:

| Model | Description |
|------|-------------|
| Linear Regression | Baseline regression model |
| Decision Tree Regressor | Tree-based model for nonlinear patterns |
| Random Forest Regressor | Ensemble model improving prediction accuracy |
| Gradient Boosting Regressor | Boosting-based model for optimized performance |

### 🏆 Best Model

Random Forest / Gradient Boosting models provided better prediction accuracy compared to basic regression models.

---

# 📈 Evaluation Metrics

Model performance was evaluated using:

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score

These metrics measure how well the model predicts customer purchase amounts.
